## File Naming Convention

Use lowercase with hyphens:
- `radboud-university.png`
- `nwo.png`
- `surf.png`
- `hpi.png`
- `dans.png`
- `vu-amsterdam.png`
- `uva.png`
- `eur.png`
- `tilburg-university.png`
- `tdcc-ssh.png`

## Recommended Sizes

- Full logos: 400px width (maintain aspect ratio)
- Icons/favicons: 64x64px or 128x128px

## Usage Notes

- Prefer SVG format when available for scalability
- PNG with transparent background for raster
- Check each organization's brand guidelines for usage rules
